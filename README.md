# WACC-Calculator-Python
WACC represents the average rate of return a company is expected to pay to all its security holders (debt and equity holders). It's the minimum return a company must earn on its existing asset base to satisfy its creditors, owners, and other providers of capital. A simple yet powerful Python script that calculates the Weighted Average Cost of Capital (WACC) - a crucial metric in corporate finance and valuation.
WACC is used for: Discounting future cash flows in DCF valuation, Investment appraisal and project evaluation, Determining the cost of capital for mergers and acquisitions, Performance benchmarking.

**How to Run This File:**

Double-click WACC Calculator.py in File Explorer

The script will run in a command window

**Prerequisites:** Python 3.x must be installed on your system.

**Input Example:**

Welcome to the WACC Calculator!

Please enter the following values:

Market value of equity ($): 50000

Market value of debt ($): 40000

Cost of equity (as decimal, e.g., 0.10 for 10%): .10

Cost of debt (as decimal, e.g., 0.05 for 5%): 0.05

Corporate tax rate (as decimal, e.g., 0.30 for 30%): .25

**Output Example:**
Your WACC is: 7.22%

Press Enter to exit...
